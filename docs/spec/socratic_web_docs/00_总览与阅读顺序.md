# 苏格拉底式智能辅学 Web 系统开发文档包（第一版）

## 1. 项目定位

本项目是一个**面向初中信息技术课堂**的局域网 Web 教学系统，采用**苏格拉底式追问**作为核心交互方法，不做模型微调，使用现成大模型 API，通过**提示词工程 + 轻量 RAG + 课堂控制 + 学习过程数据沉淀**实现课堂辅学。

系统目标不是做一个泛聊天机器人，而是做一个可在**机房课堂**中真实使用的、由教师统一组织、学生分角色登录、管理员可控配置的教学 Web 平台。

## 2. 当前已锁定的关键约束

- 主要对象：初中信息技术课堂，后续再扩高中
- 主要场景：机房可联网课堂，一个班同时使用
- 系统端口：学生端为主，教师端为辅，管理员端必需
- 教学组织：学生先选章节/任务，先看知识讲解，再进入问题链式对话
- 交互风格：系统按“引导问题链”一步步发问，学生跟着走
- 问题链生成：以“教师给大纲 + 系统补全”为主，同时支持全自动模板生成
- 内容来源：上传教材自动提取 > 根据章节标题自动生成 > 教师手工补充
- 资料处理：支持 Word/PDF、TXT/Markdown/网页文本、PPT/PPTX、图片（按优先级）
- 课堂控制：教师可一键推送全班进入同一章节/任务，关键页面尽量实时
- 结果沉淀：课堂统计分析、任务结果、学习进度、对话记录都要
- API 路线：国内大模型 API，多模型可切换
- 技术偏好：适合计算机基础较弱用户，支持 Windows 局域网一键启动
- 展示目标：不是比赛包装稿，而是能实际指导开发落地的文档包

## 5. 第一版功能边界

### 必做
- 三端角色：管理员 / 教师 / 学生
- 班级、学生账号、教师账号管理
- 教材上传与解析
- 章节、知识点、问题链、任务管理
- 学生学习页：讲解 + 问题链对话 + 代码运行 + 任务提交
- 教师课堂控制台：一键推送、实时进度、课堂统计
- 多模型配置与切换
- 轻量 RAG
- 基础安全约束
- Windows 局域网部署 + 一键启动

### 不做或延后
- 语音输入输出
- 手机端适配
- 真正复杂的在线 IDE
- 模型微调
- 深度学习画像、复杂推荐算法
- 复杂审批流
