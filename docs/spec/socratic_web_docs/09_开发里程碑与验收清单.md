# 开发里程碑与验收清单

## 1. 里程碑规划

## M1：项目骨架可启动
### 输出
- FastAPI 项目能运行
- 登录页与三端首页可访问
- 数据库初始化成功
- `start_server.bat` 可用

### 验收
- 本机访问成功
- 局域网内另一台电脑可访问首页

## M2：课堂主流程跑通
### 输出
- 教师可开始课堂
- 学生可进入当前课堂页
- 学生可提交文字回答
- 教师可看到基础进度

### 验收
- 一名教师 + 至少 2 名学生 demo 跑通

## M3：教材与内容管理跑通
### 输出
- 教材上传
- 自动解析基础文本
- 生成章节与知识点初稿
- 教师可修改保存

### 验收
- 上传一份材料后，可得到可编辑章节内容

## M4：问题链与对话闭环跑通
### 输出
- 问题链生成
- 学生端按问题链推进
- 提示层级可用
- 数据入库

### 验收
- 学生完成一条完整问题链

## M5：代码运行闭环跑通
### 输出
- 学生能提交 Python 代码
- 后端运行代码
- 返回结果
- AI 基于结果追问

### 验收
- 至少 3 个示例代码场景可稳定执行

## M6：课堂看板可用
### 输出
- 在线人数
- 完成率
- 平均进度
- 卡点分布

### 验收
- 教师在课堂中可实时看到数据变化

## M7：管理员配置可用
### 输出
- 教师/班级/学生管理
- 模型配置
- 系统参数配置

### 验收
- 不改代码即可新增模型配置并启用

## 2. 最终验收清单

### 系统运行
- [ ] Windows 本机一键启动
- [ ] 局域网多机访问成功
- [ ] 三端登录成功

### 管理员端
- [ ] 能创建教师
- [ ] 能创建班级
- [ ] 能导入学生
- [ ] 能配置多个模型
- [ ] 能修改系统设置

### 教师端
- [ ] 能上传教材
- [ ] 能得到章节与知识点初稿
- [ ] 能编辑讲解和问题链
- [ ] 能发布章节/任务
- [ ] 能一键开始课堂
- [ ] 能看到课堂实时统计
- [ ] 能点评学生

### 学生端
- [ ] 能进入当前课堂
- [ ] 能查看讲解
- [ ] 能按问题链学习
- [ ] 能提交文字答案
- [ ] 能提交并运行 Python 代码
- [ ] 能提交任务文件
- [ ] 能看到学习小结

### 模型与智能能力
- [ ] 多模型可切换
- [ ] 问题链自动生成
- [ ] 讲解自动生成
- [ ] 教材检索可用
- [ ] 安全约束可用

- 管理员登录
- 查看模型配置
- 查看班级和账号

### 演示脚本 2
- 教师登录
- 上传教材
- 自动生成章节和问题链
- 一键开始课堂

- 学生登录
- 查看讲解
- 回答问题
- 写一段 Python 代码并运行
- 收到系统追问与总结

### 演示脚本 4
- 教师查看全班实时进度和卡点
- 对某学生做点评
