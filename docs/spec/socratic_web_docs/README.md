# 追问智学文档包

这是“苏格拉底式智能辅学 Web 系统”的开发文档压缩包，重点面向：

- Windows + Python 3.11
- 局域网机房部署
- 不微调模型，只用现成 API
- 让 Codex 可直接按文档开发

## 文件说明
- `01_PRD_产品需求文档.md`：需求文档
- `02_技术架构设计.md`：架构与技术路线
- `03_角色权限_页面流程设计.md`：页面和角色设计
- `04_数据库设计.md`：数据库设计
- `05_API接口设计.md`：接口清单
- `06_提示词工程与RAG设计.md`：智能部分设计
- `07_部署与一键启动方案.md`：部署路径
- `08_Codex开发任务书.md`：交给 Codex 的任务书
- `09_开发里程碑与验收清单.md`：里程碑和验收
- `10_建议的示范课程种子数据.md`：示范课程内容
