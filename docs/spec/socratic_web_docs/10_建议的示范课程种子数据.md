# 示范课程种子数据建议（第一版）

## 1. 课程名称
初中信息技术（Python 入门示范）

## 2. 章节一：Python 的输入、输出与变量

### 2.1 章节简介
本章帮助学生认识 Python 程序最基础的输入、输出和变量概念，理解程序如何接收信息、保存信息并输出结果。

### 2.2 知识点
1. `print()` 的基本作用
2. `input()` 的基本作用
3. 变量的意义
4. 变量名与字符串字面量的区别
5. 输入、处理、输出的基本流程

### 2.3 问题链示例
1. 如果程序要和用户“交流”，最基本要做哪两件事？
2. 为什么 `input()` 得到的内容通常要先保存到变量里？
3. `print(name)` 和 `print("name")` 有什么区别？
4. 如果我把同一个变量重新赋值，会发生什么？
5. 用一句话说说“输入—处理—输出”的关系。

### 2.4 任务示例
写一个程序：
- 输入你的姓名
- 输入你的年龄
- 输出“你好，xxx，你今年 xx 岁”

## 3. 章节二：Python 条件判断（或循环结构）

### 3.1 章节简介
本章帮助学生理解程序如何根据条件做出不同选择，或如何通过循环重复执行任务。

### 3.2 知识点（条件判断版本）
1. `if` 的基本含义
2. 条件表达式的真假
3. 缩进的作用
4. `if-else` 的双分支结构
5. 条件判断的现实场景

### 3.3 问题链示例
1. 在生活中，什么时候需要“先判断再行动”？
2. 程序里的“条件成立”是什么意思？
3. 为什么 `if` 后面的代码要缩进？
4. 如果条件不成立，程序会怎样走？
5. 哪些生活场景适合用 `if-else` 表达？

### 3.4 任务示例
写一个程序：
- 输入一个分数
- 如果分数大于等于 60，输出“及格”
- 否则输出“不及格”
